We're excited to announce MagShare, a fork of the excellent BeeBEEP project! MagShare continues the mission of providing secure, serverless messaging for local area networks.

Why MagShare?

  • Continued development and maintenance of the BeeBEEP codebase
  • Modern UI improvements and updated dependencies
  • Enhanced cross-platform compatibility
  • Community-driven development approach
  • Regular security updates and bug fixes

We extend our heartfelt gratitude to Marco Mastroddi and the BeeBEEP community for creating such a robust foundation. MagShare builds upon this legacy while ensuring the project remains active and well-maintained.

üîê Security First

üìÖ January 2025 Security

MagShare inherits BeeBEEP's robust security architecture with end-to-end encryption at its core:

  • AES-256 Encryption: Military-grade encryption for all messages
  • Zero Server Architecture: Direct peer-to-peer communication means no third-party can intercept your messages
  • Random Key Generation: Unique encryption keys for each connection
  • Local Network Only: Messages never leave your trusted network

Your privacy and security are our top priorities. All communication happens directly between users without any central server that could be compromised.

üìÖ January 2025 Features

MagShare comes packed with features designed for modern workplace communication:
