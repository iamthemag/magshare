MagShare Help Guide

Getting Started

MagShare is a secure, serverless messenger designed for local area networks. This guide will help you install and use MagShare effectively.

Installation

Windows

  1. Download the latest .exe installer from the download page
  2. Run the installer and follow the on-screen instructions
  3. Launch MagShare from the Start menu or desktop shortcut

Linux (Debian/Ubuntu)

  1. Download the .deb package from the download page
  2. Install using the command:
sudo dpkg -i magshare_*.deb
sudo apt-get install -f  # Install dependencies if needed
  1. Launch MagShare from your applications menu or run magshare in terminal

macOS

  1. Download the .dmg file from the download page
  2. Open the DMG file and drag MagShare to your Applications folder
  3. Launch MagShare from Applications
  4. If prompted about security, go to System Preferences → Security & Privacy and allow MagShare to run

First Time Setup

  1. Set Your Display Name: Choose a name that will be visible to other users on your network
  2. Configure Your Status: Set your availability status (Available, Away, Busy, etc.)
  3. Network Discovery: MagShare will automatically discover other users on your local network
  4. Firewall Settings: Make sure your firewall allows MagShare to communicate on your local network

Core Features

Starting a Chat

  • Select a user from the user list
  • Double-click their name or right-click and select "Chat"
  • Type your message in the input field and press Enter

Group Chats

  • Click the "Create Group" button or use the menu option
  • Select multiple users to add to the group
  • Give your group a name
  • All group members will be notified and can participate in the conversation

File Sharing

  • Drag and drop files into any chat window, or
  • Click the attachment icon and select files to share
  • Files are transferred directly between users (peer-to-peer)
  • You can pause and resume file transfers
  • Large files are supported

Encryption

  • All messages are encrypted with AES-256 encryption by default
  • Encryption keys are unique for each connection
  • No configuration needed - it's automatic

Settings & Configuration

Network Settings

  • Port Configuration: Default port is 6257. Change if needed in Settings → Network
  • Broadcast Settings: Configure network broadcast settings if users aren't appearing
  • Connection Mode: Choose between LAN and Internet mode (if VPN is configured)

Appearance

  • Change themes and color schemes
  • Adjust font sizes for better readability
  • Customize notification sounds

Privacy

  • Enable/disable message history
  • Configure message retention period
  • Set up blocked users list

Troubleshooting

Can't See Other Users

  • Ensure all users are on the same network
  • Check firewall settings and allow MagShare through
  • Verify that the network allows broadcast/multicast traffic
  • Try manually adding users by IP address

File Transfer Issues

  • Check available disk space on receiving end
  • Ensure both users have stable network connection
  • Try smaller files first to test connectivity
  • Check firewall isn't blocking file transfers

Connection Problems

  • Restart MagShare on both ends
  • Check network connectivity
  • Verify port 6257 (or your custom port) is not blocked
  • Try disabling VPN temporarily to test

Keyboard Shortcuts

  • Ctrl/Cmd + N: New chat
  • Ctrl/Cmd + G: Create group
  • Ctrl/Cmd + F: Find/Search
  • Ctrl/Cmd + W: Close chat
  • Ctrl/Cmd + Q: Quit application
  • Ctrl/Cmd + ,: Open settings
